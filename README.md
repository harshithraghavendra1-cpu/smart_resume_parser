# smart_resume_parser

## Overview
This project extracts structured information from resumes using NLP.

## Features
- Extracts name, email, phone, skills, education, experience
- Supports PDF and DOCX
- Streamlit UI
- Outputs JSON and CSV

## Technologies
- Python
- spaCy
- PyMuPDF
- python-docx
- Streamlit

## How to Run
pip install -r requirements.txt  
python -m spacy download en_core_web_sm  
python -m streamlit run app.py

## Output
Structured JSON and CSV files
